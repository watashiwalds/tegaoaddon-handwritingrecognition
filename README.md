### Tegao - Japanese Word Helper
#### Optional Addon: Handwriting Recognition
Tegao: HandwritingRecognition is an optional app acts as a service that hosts the offline TFLite model that able to recognize >3000 character (hiragana & kanji), based on ETL9G dataset.

Required [Tegao](https://github.com/watashiwalds/tegao-japanesewordhelper) to be installed to use.

Build on collaboration with [@ducvinh101101](https://github.com/ducvinh101101) who train the model used in this addon.
